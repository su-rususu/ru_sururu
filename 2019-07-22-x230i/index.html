x230iを買ってた話。

投稿日 2019-07-22 編集日 2020-01-22 カテゴリ PC 、 買い物・貰い物

理由的な

  • 前々から(前のOSC辺りから)外で書いたり雑に時間を潰す目的なやつがほしいなと思っていたので。

  • SSDが入るのであればそこまで高スペック最新なものでもなくていいと思っていたので中古の中から探すことに。後々のメンテ性を考えてThinkpadシリーズと考えていました。

ブツ

  • Thinkpad x230i 230677Jを買いました。約2万円。

    • CPU - Core-i3 3100M / MEM - 6GB

    • OSなしの形態で販売されていました。

    • SSD 250GBに換装。

    • 搭載されていたのが日本語キーボードだったため、USキーボードをAmazonでポチって交換。交換方法はこの辺りで。

OS

  • Windows 7のリカバリーメディアが付属していたのでそこから復旧後にWindows 10 Proにアップグレードしました。

    • やはり解像度が低いこともありフォントレンダリングまわりに不満がありますがM+ 1c Unhintedを使うことで許容できるレベルに。

    • ノベルゲームなどであればゆっくり寝転びながらできるのでWindowsで使用しています。最近Steamでも買えるし。

しばらく使用してみて

  • 1.5kgはまぁまぁ重たさを感じる。最新のものが軽いだけってのはあるけど

  • 買ってから気づいたけどFHDにするのがかなりめんどい。ので次はFHD機種を狙いたい。

  • Bluetoothなしのモデルでしたが今の所使うデバイスはないので困ってないです。

  • キーボードが打ちやすいね。

いい感じのものでした。バッテリーを購入して交換したら更に良くなりそう。
